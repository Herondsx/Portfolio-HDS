Plataforma de Gestão de Projeto Mecânico

Sistema completo para gerenciamento de projetos de engenharia mecânica, controle de milestones e rastreamento de dispositivos em tempo real.

Funcionalidades principais:

  • Gerenciamento detalhado de milestones e cronogramas de projeto.
  • Cadastro e controle granular de dispositivos mecânicos com informações técnicas.
  • Sistema de controle de status a nível de dispositivo (aprovado, em revisão, pendente, etc.).
  • Dashboard interativo com métricas em tempo real e visualização de progresso.
  • Sistema de notificações e alertas para prazos e mudanças de status.
  • Geração de relatórios customizados e exportação de dados.
  • Controle de versões e histórico de alterações de cada dispositivo.

Stack tecnológica: React + Vite, Node.js, PostgreSQL, REST APIs.

WeldScanner

Ferramenta desktop de análise e desconstrução completa de backups de códigos de robôs industriais.

Funcionalidades principais:

  • Parsing inteligente de backups de código de robôs (múltiplos formatos).
  • Extração e desconstrução completa de pontos de solda, trajetórias e parâmetros.
  • Visualização estruturada de dados com interface intuitiva em WPF.
  • Validação automática de integridade e detecção de inconsistências.
  • Exportação para CSV/Excel e integração com sistemas internos.
  • Sistema de busca avançada e filtros customizáveis.

Stack tecnológica: WPF, C#, XAML, .NET Framework, Visual Studio Professional 2022.
